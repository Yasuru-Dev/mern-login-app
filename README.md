# MERN Login & Register App

A full-stack authentication app built with the MERN stack.

## Features
- User Registration
- User Login
- Password Encryption
- JWT Authentication

## Technologies Used
- **MongoDB** - Database
- **Express.js** - Backend framework
- **React.js** - Frontend library
- **Node.js** - Runtime environment
- **JWT** - Authentication tokens
- **bcryptjs** - Password encryption
- **React Router** - Page navigation
- **Axios** - HTTP requests

## Getting Started

### Run the frontend:
cd client
npm run dev


### Run the backend:
cd server
npm run dev
